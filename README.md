# Core Java Array Programs

A collection of fundamental Java programs demonstrating various array manipulation techniques and concepts.

## Programs Included

### Sorting Operations
- **ArraySortUsingBuiltInmethod.java** - Sort array elements using built-in methods like Arrays.sort()
- **ArraySortWithoutUsingBuiltInMethod.java** - Sort array elements using custom sorting logic (bubble sort, etc.)

### Finding Elements
- **LargestNumberInAnArray.java** - Find the maximum element in an array
- **SmallestNumberInAnArray.java** - Find the minimum element in an array
- **SecondMaximumElement.java** - Find the second largest element in an array
- **SecondMinimumElement.java** - Find the second smallest element in an array

### Array Filtering
- **EvenNumberInAnArray.java** - Extract and display even numbers from an array
- **OddNumberInEvenIndex.java** - Find odd numbers at even index positions
- **ElementsInEvenIndex.java** - Extract elements at even indices

### Array Modifications
- **InsertElementAtSpecificIndex.java** - Insert an element at a specific array position
- **DeleteAnElementFromAnArray.java** - Remove an element from an array
- **RemoveDuplicatesFromAnArray.java** - Eliminate duplicate elements
- **ReversingArray.java** - Reverse array elements (various approaches)
- **ReversingArrayUsingNewArray.java** - Reverse array by creating a new array
- **ReversingArrayWithoutUsingNewArray.java** - Reverse array in-place without extra space

### Core Concepts
- **SumOfArray.java** - Calculate the sum of all array elements
- **PassingArrayReferenceToAMethod.java** - Demonstrate passing arrays to methods and reference behavior

### Exception Handling
- **AIOBE.java** - ArrayIndexOutOfBoundsException examples
- **NPE.java** - NullPointerException examples

## Getting Started

Each Java file contains a standalone program demonstrating a specific array concept. You can compile and run any file individually:

```bash
javac FileName.java
java FileName
```

## Learning Objectives

These programs cover essential Java array operations:
- Array creation and initialization
- Iteration and traversal
- Sorting and searching
- Array manipulation (insert, delete, reverse)
- Handling array exceptions
- Reference passing and memory concepts

## Prerequisites

- Java Development Kit (JDK) installed
- Basic understanding of Java syntax
- Familiarity with command-line compilation and execution
